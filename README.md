## Development Setup

1. Clone the repository:
```bash
git clone [repository-url]
cd electron-video-player
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Build

To build the application:
```bash
npm run build
```
